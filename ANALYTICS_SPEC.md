# Analytics Specification

## Metrics

- total items acquired
- acquisitions per month
- acquisitions by collection area
- usage (checkouts)
- recommendation counts
- approval rates
- pending items

## Targets

- monthly acquisition target
- percentage targets by collection area

## Reports

The system must generate statements like:

"In the last 60 days, Manga/Anime had the highest circulation."

All such statements must be:
- data-derived
- reproducible
- traceable

## Collection Balance

The system must compare:
- internal distribution
- external field baseline (if available)

Differences must be explicitly calculated.
